Aplikační software

Co je aplikační software

Aplikační software jsou uživatelské programy určené pro jeden nebo více specifických účelů. Pro jejich spouštění je nezbytná přítomnost základního software - operačního systému.

Základní skupiny aplikací

Kancelářské balíky

Kancelářské balíky tvoří symbolický „balík“ programů, který obvykle obsahuje:

  • textový procesor
  • tabulkový procesor
  • nástroj pro tvorbu prezentací
  • databázový systém
  • groupware nástroje (organizace času, schůzek)
  • grafické editory

Mezi nejznámější patří Microsoft Office a Apple iWork. Zdarma jsou dostupné například OpenOffice nebo WPS Office.

Grafické aplikace

Grafické programy umožňují tvorbu grafických dokumentů, skenování předloh nebo zpracování fotografií. Dominantní postavení má profesionální software firmy Adobe (Photoshop, Illustrator).

Mezi oblíbené open-source aplikace patří:

  • GIMP - rastrová grafika
  • Inkscape - vektorová grafika

Speciální oblasti tvoří programy pro animace, konstrukční systémy CAD (Computer Aided Design) a programy DTP (Desktop Publishing) pro tvorbu tiskovin.

Multimediální programy

Multimediální software slouží k vytváření, úpravě a prezentaci audiovizuálních dat. Patří sem:

  • multimediální přehrávače
  • programy pro editaci zvuku a videa
  • nástroje pro tvorbu multimediálních prezentací
  • výukové programy
  • počítačové hry

Programy pro komunikaci v síti

Tyto programy slouží ke správě sítí, vzdálenému řízení počítače, přenosu souborů a připojení podnikových sítí k Internetu.

K základním klientským aplikacím patří:

  • webové prohlížeče
  • e-mailoví klienti
  • groupware (software pro týmovou spolupráci)
  • online komunikace (IM - Instant Messaging)

Informační a databázové systémy

Informační systémy (IS) představují komplexní software určité firmy nebo instituce. Základem bývají datové sklady (data warehouse) založené na databázových systémech.

Podnikové informační systémy označujeme jako ERP (Enterprise Resource Planning). Obsahují moduly pro:

  • výrobu
  • logistiku
  • správu majetku
  • účetnictví

Mezi hlavní výrobce IS a databází patří Oracle, Microsoft, IBM a SAP.

Malware a antivirové programy

Malware je škodlivý software určený k napadení nebo poškození systému. Antivirové programy (např. AVG, AVAST, ESET, Kaspersky) chrání počítač před těmito hrozbami.

Integrovaná vývojová prostředí

IDE (Integrated Development Environment) používají programátoři pro vývoj aplikací. Obsahují:

  • editor kódu
  • ladicí nástroje
  • překladače
  • další nástroje pro efektivní vývoj
